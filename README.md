# thailandpost-php-client
A hacky workaround PHP client library for D Post inter of Thailand post (Thai post)


Requirement
```
ext-json
ext-fileinfo
ext-gd
```

Install
```
composer install
```

Configuration
```
- cp .env.example .env
- Update login Username / Password in the .env file
```


Run an example usage

```
php example.php
```
